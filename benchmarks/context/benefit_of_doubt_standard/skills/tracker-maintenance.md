# Skill: Tracker Maintenance

Use this procedure when the user gives messy notes that imply tracker cleanup.

- Identify the operative tracker changes and ignore unrelated clutter.
- Preserve existing tracker items unless the user explicitly changes them.
- If a tracker item wording changes, update the existing item.
- If a genuinely new task is implied, add it.
- After changes, report the current tracker state.
- Do not confuse tracker state with file-based task lists.
