# Skill: Handoff Artifact

Use this procedure when the user asks for a handoff, rollout, or compact operator artifact.

- Gather the minimum authoritative inputs needed for the artifact.
- Keep the artifact compact and operator-readable.
- Prefer short sections and grounded facts.
- Distinguish pending work from risks.
- If the user later asks what is currently in the artifact, inspect the file rather than relying on memory.
